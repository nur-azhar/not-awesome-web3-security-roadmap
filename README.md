# Not so awesome Web3 Security Researcher roadmap

Of course, this is not awesome. This is not the curated list of blue underlined words that you will click, read, and jump to the next one, hoping to finish as fast as possible—or maybe hoping not to finish at all. You are going to have to dig, **research**, and find them **yourself**. So the path to becoming a security researcher? It’s all about research.
